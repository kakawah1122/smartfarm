## 一、核心问题修正 ⚠️

### 1.1 疾病名称混淆（致命错误）

**问题**：系统将"鹅副粘病毒病"误标为"鹅瘟"，导致诊断和治疗方案错误。

**修正**：
```
✅ 小鹅瘟 = 真正的"鹅瘟"
   • 病原：鹅细小病毒（Goose Parvovirus）
   • 日龄：1-20天高发
   • 特征：肠道黏膜内侧白色假膜（诊断金标准）

✅ 鹅副粘病毒病 ≠ 鹅瘟
   • 病原：禽副粘病毒I型（APMV-1）
   • 日龄：30-90天高发
   • 特征：神经症状+消化道出血（无假膜）

✅ 鸭瘟 ≠ 鹅瘟
   • 病原：鸭瘟疱疹病毒
   • 日龄：成年鹅多发
   • 特征：食道和泄殖腔特征性病变
```

**影响范围**：
- `cloudfunctions/ai-diagnosis/disease-knowledge.js` - 疾病知识库
- `cloudfunctions/ai-diagnosis/index.js` - System Prompt

## 二、疾病知识库全面升级

### 2.1 新增狮头鹅生长参数

添加了狮头鹅各日龄标准体重数据，用于准确计算用药剂量：

| 日龄 | 体重范围 | 用途 |
|-----|---------|------|
| 1-7天 | 0.15-0.25kg | 雏鹅用药剂量计算 |
| 8-14天 | 0.4-0.6kg | 早期用药剂量 |
| 15-21天 | 0.8-1.2kg | 中期用药剂量 |
| 22-30天 | 1.5-2.5kg | 育成期用药剂量 |
| 31-45天 | 3.0-4.5kg | 快速生长期 |
| 46-60天 | 5.0-7.0kg | 育成后期 |
| 61-90天 | 7.5-10kg | 出栏前期 |
| 成年鹅 | 公10-12kg / 母9-10kg | 成年用药剂量 |

### 2.2 完善主要疾病的诊疗方案

优化了**16种**高发疾病的诊断标准和用药方案：

#### A. 病毒性疾病（3种）
1. **小鹅瘟** ⭐⭐⭐⭐⭐
   - 添加了完整的病原学描述
   - 强化了肠道假膜的鉴别诊断
   - 补充了预防方案（母源抗体+疫苗）

2. **鹅副粘病毒病** ⭐⭐⭐⭐
   - ⚠️ 明确标注"非鹅瘟"
   - 补充了权威用药方案：
     * 紧急免疫：I号/II号灭活疫苗，1ml/只
     * 抗体治疗：高免卵黄抗体，0.5-1ml/kg
     * 继发感染控制：头孢曲松钠20-30mg/kg
   - 添加了详细的鉴别诊断要点

3. **鸭瘟** ⭐⭐⭐
   - ⚠️ 明确标注"非鹅瘟"
   - 强调食道和泄殖腔病变的特征性

#### B. 细菌性疾病（5种）
4. **大肠杆菌病** ⭐⭐⭐⭐
   - 详细描述"三炎"并发症（心包炎+肝周炎+气囊炎）
   - 权威用药方案：
     * 一线：头孢噻呋20-40mg/kg
     * 二线：恩诺沙星10-20mg/kg
     * 三线：阿莫西林+克拉维酸25mg/kg
   - ⚠️ 明确标注禁用药物（氯霉素、呋喃类）

5. **鹅鸭疫里默氏杆菌病** ⭐⭐⭐⭐
   - 强调脑膜炎和关节炎的特征性
   - 权威用药方案：
     * 一线：头孢噻呋40mg/kg
     * 二线：氟苯尼考30mg/kg
     * 疫苗：多价灭活苗，7-10日龄首免

6. **沙门氏菌病** ⭐⭐⭐
   - 详细描述白痢症状
   - 补充了人畜共患病警示

7. **禽霍乱** ⭐⭐⭐
   - 区分最急性、急性、慢性型
   - 权威用药方案：
     * 一线：青霉素G 5-10万IU/kg
     * 疫苗：氢氧化铝灭活苗，每年2次

8. **曲霉菌病** ⭐⭐
   - 补充了真菌感染的详细用药：
     * 制霉菌素10万IU/kg
     * 伊曲康唑5-10mg/kg
   - 强调预防为主

#### C. 寄生虫病（2种）
9. **球虫病** ⭐⭐⭐
   - 区分肠球虫和肾球虫
   - 权威用药方案：
     * 一线：地克珠利1mg/kg
     * 二线：磺胺氯吡嗪钠100mg/kg
     * 预防：地克珠利预混剂，每月1次

10. **绦虫/吸虫病** ⭐⭐
    - 区分绦虫病和吸虫病
    - 用药方案：
      * 吡喹酮10-20mg/kg（绦虫）
      * 硫双二氯酚100-200mg/kg（吸虫）

#### D. 营养代谢病（3种）
11. **维生素E/Se缺乏症**（白肌病、渗出性素质）
    - 详细症状和剖检特征
    - 治疗：VE 5-10mg/kg + 亚硒酸钠0.1mg/kg

12. **维生素D/Ca/P缺乏症**（佝偻病、软骨病）
    - ⚠️ 强调"无感染性病变"是重要鉴别点
    - 治疗：VD3 1000-2000IU/只

13. **维生素B1缺乏症**（脚气病）
    - 角弓反张+抽搐
    - 治疗：VB1 10-20mg/只

14. **痛风**
    - 区分关节型和内脏型
    - 详细的发病原因和治疗方案

### 2.3 强化鉴别诊断逻辑

添加了**8大误诊陷阱**警示系统：

#### 陷阱1：疾病名称混淆（致命错误）
```
❌ 鹅副粘病毒病 = 鹅瘟
✅ 小鹅瘟 ≠ 鹅副粘病毒病 ≠ 鸭瘟
```

#### 陷阱2：仅凭单一症状诊断
```
❌ 看到拉稀就诊断肠炎
✅ 结合日龄+剖检+病程+群体发病规律
```

#### 陷阱3：忽略日龄因素
```
日龄诊断口诀：
• 0-7天 → 小鹅瘟、脐炎、曲霉菌
• 8-21天 → 大肠杆菌、沙门氏菌、营养缺乏
• 22-45天 → 副粘病毒、球虫、里默氏杆菌
• 46天+ → 禽霍乱、鸭瘟、寄生虫
```

#### 陷阱4：混淆相似病变
**纤维素假膜位置鉴别**：
- 小鹅瘟：肠黏膜内侧，白色
- 大肠杆菌：浆膜外侧（心包、肝包膜），黄色
- 鸭瘟：食道和泄殖腔，出血性

**神经症状鉴别**：
- 副粘病毒：扭颈+拉绿色稀便
- 里默氏杆菌：共济失调+眼鼻分泌物+关节炎
- VE缺乏：共济失调+皮下水肿，⚠️无消化道症状
- VB1缺乏：角弓反张+抽搐，⚠️无发热无感染

**肝脏坏死灶鉴别**：
- 小鹅瘟：白色坏死灶，数量中等
- 沙门氏菌：灰白色，脾脏肿大明显
- 禽霍乱：针尖大小，数量多，散在分布
- 鸭瘟：灰黄色，较大

#### 陷阱5：忽视继发感染
- 病毒性疾病后→大肠杆菌、沙门氏菌
- 球虫病后→肠炎
- 应激后→大肠杆菌病

#### 陷阱6：忽略环境和管理因素
- 潮湿→球虫、曲霉菌
- 密度大→大肠杆菌、呼吸道病
- 温度骤变→应激、大肠杆菌
- 饲料霉变→曲霉菌、黄曲霉毒素中毒

#### 陷阱7：忽略品种特异性（狮头鹅）
- 体型大（10-12kg），用药剂量需按实际体重计算
- 生长快，易发代谢病
- 应激敏感
- 关节负重大，里默氏杆菌病关节炎更明显

#### 陷阱8：禁用药物（法律风险）
```
❌ 国家禁用：
   • 氯霉素、呋喃类、孔雀石绿、洛硝哒唑、氨苯砜

⚠️ 限制使用（休药期长）：
   • 氟喹诺酮类（恩诺沙星）：28天
   • 磺胺类：28天

✅ 推荐使用：
   • 头孢类（头孢噻呋、头孢曲松）：7天
   • β-内酰胺类（阿莫西林、青霉素）：5-7天
   • 氟苯尼考：5-7天
```

2. 症状系统分析鉴别表

**优化示例**：
```
第二步：症状系统分析（按体系分类）
→ 神经系统症状鉴别：
  • 扭颈+拉绿色稀便 → 鹅副粘病毒病（⚠️非鹅瘟！）
  • 共济失调+眼鼻分泌物+关节炎 → 里默氏杆菌病
  • 角弓反张+抽搐，无消化道症状 → VB1缺乏
  • 共济失调+皮下水肿，无消化道症状 → VE/Se缺乏
→ 消化系统症状鉴别：
  • 白色水样稀便+1-20日龄 → 小鹅瘟（⚠️真正的鹅瘟）
  • 血便或番茄酱样便 → 球虫病
  • 黄绿色稀便+呼吸困难 → 大肠杆菌病
  • 白色糊状便+肛门粘便 → 沙门氏菌病

2. 特征性病变识别（诊断金标准）
3. 假膜位置鉴别强化

**优化示例**：
```
第四步：图像证据分析（如有剖检照片）

  • ★★★ 小肠黏膜内侧白色假膜 = 小鹅瘟（⚠️真正的鹅瘟）
  • ★★★ 黄色纤维素心包炎+肝周炎+气囊炎 = 大肠杆菌"三炎"
  • ★★★ 食道和泄殖腔出血溃疡假膜 = 鸭瘟（⚠️非鹅瘟）
  • ★★ 脑膜充血+消化道出血 = 鹅副粘病毒病（⚠️非鹅瘟）
→ ⚠️ 重点检查：假膜位置（黏膜内侧vs浆膜外侧）是鉴别诊断的关键！

## 四、用药方案权威化

### 4.1 所有用药方案包含的要素

✅ **完整的用药信息**：
1. 药物通用名（非商品名）
2. 剂量（mg/kg体重 或 IU/kg体重）
3. 给药途径（口服/饮水/肌肉注射/皮下注射）
4. 给药频次（每日几次）
5. 疗程天数
6. 休药期（屠宰前停药天数）
7. 配伍禁忌（如有）
8. 注意事项

### 4.2 用药方案示例

**鹅副粘病毒病**：
```
⚠️ 权威用药方案：
- 紧急免疫：鹅副粘病毒病灭活疫苗（I号或II号），1ml/只，肌肉注射
- 抗体治疗：高免卵黄抗体，0.5-1ml/kg体重，皮下注射，连用3天
- 继发感染控制：头孢曲松钠20-30mg/kg体重，肌肉注射，每日2次，连用5-7天
- 或：阿莫西林25-50mg/kg体重，饮水给药，连用5-7天
- 支持疗法：口服补液盐+多维电解质，5g/L饮水，自由饮用
- 休药期：头孢类7天，阿莫西林5天
```

**大肠杆菌病**：
```
⚠️ 权威用药方案（需药敏试验！）：
- 一线：头孢噻呋20-40mg/kg体重，肌肉注射，每日1次，连用3-5天
- 二线：恩诺沙星10-20mg/kg体重，肌肉注射或饮水，每日1次，连用5-7天
- 三线：阿莫西林+克拉维酸25mg/kg体重，饮水给药，连用5-7天
- 配合：多维电解质+益生菌，改善肠道菌群
- 环境：彻底消毒（聚维酮碘、过硫酸氢钾），降低饲养密度
- 休药期：头孢类7天，氟喹诺酮类28天（⚠️重要）
- ⚠️ 禁用：氯霉素、呋喃类（国家禁用）
```

## 五、优化效果预期

### 5.1 准确率提升
- **疾病鉴别准确率**：通过强化日龄定位+症状鉴别+剖检特征，预计提升20-30%

- **日龄分段诊断**：根据0-7天、8-21天、22-45天等不同日龄，快速定位高风险疾病
- **环境管理建议**：结合温湿度、饲养密度、饲料等环境因素，提供综合防控方案
