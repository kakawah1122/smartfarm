# 微信小程序隐私政策配置

## 配置位置
微信公众平台 → 设置 → 功能设置 → 用户隐私保护指引

---

## 一、收集的用户信息

### 1. 用户昵称和头像
- **收集方式**：用户主动授权
- **使用目的**：展示用户个人信息、区分不同用户
- **是否必须**：是

### 2. 位置信息
- **收集方式**：用户主动授权
- **使用目的**：获取当地天气信息，为养殖管理提供参考
- **是否必须**：否

---

## 二、隐私保护指引文本（复制使用）

```
【鹅数通】小程序隐私保护指引

一、我们收集的信息

1. 用户昵称和头像
   - 用途：展示您的个人信息，方便识别账户
   - 收集方式：需要您主动授权

2. 位置信息
   - 用途：获取当地天气数据，为养殖管理提供参考
   - 收集方式：需要您主动授权
   - 说明：您可以拒绝授权，不影响其他功能使用

二、信息的使用

我们收集的信息仅用于：
- 展示您的用户信息
- 提供本地化天气服务
- 记录和管理您的养殖生产数据
- 改进我们的服务

三、信息的存储

- 您的数据存储在微信云开发平台
- 我们采用加密措施保护您的数据安全
- 我们不会将您的数据分享给第三方

四、您的权利

- 您可以随时查看和管理您的个人信息
- 您可以通过「我的 → 隐私设置」管理授权
- 您可以联系我们删除您的数据

五、联系我们

如有隐私相关问题，请通过小程序内的反馈功能联系我们。

更新日期：2025年11月
```

## 三、在微信公众平台的配置步骤

### 步骤1：进入隐私设置
1. 登录 mp.weixin.qq.com
2. 点击左侧「设置」
3. 选择「功能设置」
4. 找到「用户隐私保护指引」

### 步骤2：配置收集信息
勾选以下选项：
- [x] 用户信息（昵称、头像）
- [x] 位置信息

### 步骤3：填写使用说明
对每个勾选项填写使用目的：

| 信息类型 | 使用说明 |
|---------|---------|
| 用户信息 | 用于展示用户个人信息和账户识别 |
| 位置信息 | 用于获取当地天气数据，为养殖管理提供参考 |

### 步骤4：提交审核
点击「提交」，等待审核通过

## 四、代码中的隐私相关配置

### 4.1 app.json 权限声明（已配置）
```json
{
  "permission": {
    "scope.userLocation": {
      "desc": "您的位置信息将用于获取当地天气信息"
    }
  },
  "requiredPrivateInfos": [
    "getLocation"
  ]
```

## 五、注意事项

1. **个人账号限制**：部分敏感权限可能不可用
2. **必须配置**：2024年起，隐私政策是审核必查项
3. **如实填写**：确保配置与实际使用一致
4. **定期更新**：功能变化时及时更新隐私政策

---

**配置完成后**，请在微信开发者工具中测试隐私授权流程是否正常。
