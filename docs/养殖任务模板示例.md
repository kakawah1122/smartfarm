# 养殖任务模板示例

## Excel 模板格式说明

### 支持的列名（中英文均可）
- **日龄** / Day Age - 任务开始的日龄
- **任务名称** / Task Name - 任务的标题
- **任务描述** / Description - 任务的详细说明
- **任务类型** / Type - 任务类型（健康检查/疫苗接种/用药管理等）
- **任务分类** / Category - 任务分类
- **优先级** / Priority - 高/中/低
- **用量** / Dosage - 用药或饲料的用量说明
- **持续天数** / Duration - 任务持续的天数
- **所需材料** / Materials - 需要的材料清单
- **备注** / Notes - 其他说明
- **预计用时** / Estimated Time - 预计耗时（分钟）

### 智能任务分解功能

系统会自动识别并分解以下格式的任务：

#### 1. 通过"持续天数"列指定
| 日龄 | 任务名称 | 持续天数 | 说明 |
|------|----------|----------|------|
| 1 | 开口药 | 3 | 系统会自动分解为第1、2、3天的任务 |
| 5 | 疫苗接种 | 1 | 单天任务，不分解 |

#### 2. 从任务名称中自动识别
系统能识别以下关键词：
- **连喂X天**：如"开口药连喂3天"
- **连续X天**：如"维生素连续5天"
- **持续X天**：如"抗生素持续7天"
- **X天疗程**：如"5天疗程"
- **连用X天**：如"益生菌连用3天"
- **第X-Y天**：如"第1-3天用药"

#### 3. 分解后的任务展示
原始任务：
- 第1日龄：开口药（连喂3天）

分解后：
- 第1日龄：开口药（首日）
- 第2日龄：开口药（第2天）
- 第3日龄：开口药（末日）

### 使用建议

1. **标准化命名**：使用统一的任务名称，便于后续管理
2. **明确持续时间**：对于连续性任务，明确标注持续天数
3. **合理分配优先级**：疫苗和关键用药设为高优先级
4. **详细记录用量**：便于实际操作时参考
5. **利用备注**：记录特殊注意事项

### 导入步骤

1. 准备Excel文件（支持.xlsx和.xls格式）
2. 在任务管理页面点击"导入模板"
3. 选择"从文件导入"
4. 选择准备好的Excel文件
5. 系统会自动解析并显示识别结果
6. 查看任务分解预览
7. 确认无误后导入

### 注意事项

- 系统会自动过滤掉没有任务名称的空行
- 日龄必须是数字，默认为1
- 持续天数默认为1天
- 优先级默认为"中"
- 任务类型默认为"饲养管理"
