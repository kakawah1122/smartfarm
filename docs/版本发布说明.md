## 审核说明（提交时使用）

```
【小程序名称】鹅数通

【功能简介】
鹅数通是一款专为养鹅农户设计的智能养殖管理工具，提供生产管理、健康管理、财务分析等一站式服务。

【主要功能】
1. 生产管理：批次入栏/出栏记录、物料管理、投喂记录
2. 健康管理：疫苗接种、治疗记录、死亡登记、AI诊断
3. 数据分析：成本分析、收益统计、健康报表

【测试账号】
如需测试，请使用微信扫码登录即可体验全部功能。

【其他说明】
- 本小程序为个人开发，用于农业养殖管理
- 不涉及任何支付交易功能
- 所有数据存储在微信云开发平台
```

## 上传步骤

### 1. 微信开发者工具上传
```
1. 打开微信开发者工具
2. 确保项目编译无错误
3. 点击右上角「上传」按钮
4. 填写版本号：1.0.0
5. 填写项目备注：首次发布版本
6. 点击「上传」
```

### 2. 微信公众平台提交审核
```
1. 登录 mp.weixin.qq.com
2. 进入「版本管理」
3. 找到刚上传的版本
4. 点击「提交审核」
5. 按提示填写信息
6. 提交等待审核
```

## 常见审核问题及解决

### 问题1：类目不符
**解决**：选择「工具 → 信息查询」或「生活服务 → 生活服务」类目

### 问题2：隐私政策未配置
**解决**：在公众平台配置用户隐私保护指引

### 问题3：功能不完整
**解决**：确保所有页面功能可用，无空白页面

### 问题4：存在测试内容
**解决**：清除所有测试数据和调试代码
