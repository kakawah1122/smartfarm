# 虚拟列表测试检查清单

## 📱 测试准备
- [ ] 编译并运行小程序
- [ ] 确保有至少50条以上的财务记录（越多越好）
- [ ] 打开微信开发者工具的Performance面板

## 🚀 快速测试路径
1. 首页 → 财务管理 → 查看全部 → 财务记录
2. 右下角看到黑色测试按钮："虚拟渲染：关"

- [ ] 滚动正常

- [ ] 点击按钮切换到"开"

- [ ] 滚动更加流畅
- [ ] 快速滚动到底部，数据完整

## ⚠️ 问题检查

### 如果数据不显示
1. 检查控制台是否有错误
2. 尝试关闭再开启虚拟渲染
3. 刷新页面重试

### 如果滚动卡顿
1. 减少测试数据量
2. 检查是否有其他性能问题
3. 在真机上测试
