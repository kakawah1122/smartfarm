## 一、服务内容

### 1.1 服务描述

「鹅数通」是一款面向养殖户的数据管理工具小程序，主要提供以下服务：

1. **健康管理**
   - 鹅群治疗记录管理
   - 鹅群治愈记录管理
   - 鹅群死亡记录管理
   - 健康数据统计和展示

2. **养殖待办**
   - 待办任务创建和管理
   - 任务完成状态跟踪
   - 任务提醒功能

3. **数据管理**
   - 鹅价信息记录和查询
   - 报销记录管理
   - 财务数据统计

4. **AI辅助功能**
   - AI疾病诊断
   - AI财务分析
   - AI鹅价识别

5. **其他功能**
   - 数据统计和可视化
   - 信息查询
   - 记录备忘

### 1.2 服务性质

本小程序为免费工具服务，旨在帮助养殖户更便捷地管理养殖数据。
