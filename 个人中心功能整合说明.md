# 个人中心功能整合说明

## 📋 整合前后对比

### 原管理功能（8个）
1. 用户管理
2. 角色权限
3. 数据报表
4. 报销审批
5. 财务报表
6. 成本分析
7. 性能分析
8. 数据导出

### 整合后管理功能（3个）

#### 1. 人员管理
- **包含功能**：用户管理、角色权限
- **跳转页面**：`/packageUser/user-management/user-management`
- **说明**：整合用户管理和角色权限管理功能

#### 2. 财务管理
- **包含功能**：财务报表、报销审批、成本分析
- **跳转页面**：`/packageFinance/finance/finance`
- **说明**：整合所有财务相关功能，可通过标签页切换

#### 3. 数据分析
- **包含功能**：数据报表、性能分析
- **跳转页面**：`/packageFinance/performance-analysis/performance-analysis`
- **说明**：整合数据报表和性能分析（原本就是同一个页面）

## 🎨 界面优化

### 布局调整
- **管理功能卡片**：从 4 列布局改为 3 列布局
- **每个功能项**：显示主标题 + 副标题（描述包含的功能）
- **间距优化**：增加了功能项之间的间距，提升可读性

### 样式改进
- 主标题字体加粗（`font-weight: 600`）
- 副标题使用灰色小字显示（`font-size: 20rpx, color: #999`）
- 功能项内容垂直排列（`flex-direction: column`）

## 📱 用户体验提升

1. **功能归类更清晰**：相关功能整合到一起，降低认知负担
2. **减少选择困惑**：从 8 个选项减少到 3 个，决策更快速
3. **描述更明确**：通过副标题说明包含的具体功能
4. **布局更合理**：3 列布局在移动端显示更佳

### 导航参数传递
可以通过 URL 参数来指定默认显示的标签页：
```javascript
// 示例：跳转到财务管理的报销审批标签
wx.navigateTo({
  url: '/packageFinance/finance/finance?tab=reimbursement'
})
```
