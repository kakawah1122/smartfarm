# 云函数超时配置说明

## 问题
AI识别一直3秒超时，因为云函数默认超时时间是3秒。

## 解决方案

### 方法1：在云控制台手动设置（推荐）

1. **打开微信开发者工具**
   - 点击顶部菜单栏：云开发 → 云控制台

2. **进入云函数配置**
   - 左侧菜单：云函数
   - 找到 `recognizeGoosePrice` 云函数
   - 点击函数名称进入详情

3. **修改超时时间**
   - 找到"函数配置"或"基础配置"
   - 找到"超时时间"设置
   - 修改为：**60秒**
   - 点击"保存"

4. **同样配置 `ai-multi-model`**
   - 找到 `ai-multi-model` 云函数
   - 同样设置超时时间为 **60秒**

### 方法2：通过命令行配置（如果支持）

```bash
# 在项目根目录执行
tcb fn code update recognizeGoosePrice --timeout 60
tcb fn code update ai-multi-model --timeout 60
```

## 验证

配置完成后：
1. 重新编译小程序
2. 测试AI识别功能
3. 应该能看到完整的识别过程（15-35秒）

## 注意事项

- config.json 中的 `timeout` 单位是**秒**
- wx.cloud.callFunction 中的 `timeout` 单位是**毫秒**
- 两者都需要设置，但最终以云控制台配置为准
